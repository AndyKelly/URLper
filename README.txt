/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
			Readme: URLper             
/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

URLper is a chrome plugin which manages your bookmarks. URLper is a tool used for 
bringing you back to the content you would like to consume at some stage but not 
right now & don’t necessarily want cluttering up your browser once you've seen. 

URLper is written in Html 5, JavaScript, CSS & Json
For more info check out: http://andyjkelly.com/abouturlper/

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
			Installation Instructions            
/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

A more flushed out version of these instruction's can be found here:
http://andyjkelly.com/hot-to-use-urlper/  

To install URLper from an unpacked directory, Developer mode must be enabled

	Click the Chrome menu button on the browser toolbar. 
	Click Tools.
	Select Extensions.
	Enable the Developer Mode checkbox.

The next step is to point Chrome towards the extension.

	Click Load unpacked extension
	Navigate to the extracted URLper folder and select it
	Click OK

Just make sure the Enabled check box is ticked and you should be good to go.
